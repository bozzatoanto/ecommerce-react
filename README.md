# Ecommerce React

Proyecto front end de e-commerce desarrollado con React como práctica de maquetado, componentes reutilizables, manejo de estado y navegación entre vistas.

## Estado del proyecto

🚧 En progreso

Actualmente el proyecto se encuentra en desarrollo y sigue sumando funcionalidades para mejorar la experiencia de usuario y la lógica del carrito.

## Funcionalidades implementadas

- Catálogo de productos
- Filtro por categoría
- Buscador de productos
- Vista de detalle de producto
- Galería de imágenes por producto
- Carrito lateral desplegable
- Agregar productos al carrito
- Aumentar y disminuir cantidades
- Eliminar productos del carrito
- Navegación entre catálogo y detalle con React Router

## Próximas mejoras

- Persistencia del carrito con localStorage
- Mejoras visuales y animaciones
- Mayor variedad de imágenes por producto
- Responsive más pulido
- Feedback visual al agregar productos
- Posible integración de checkout

## Tecnologías utilizadas

- React
- JavaScript
- CSS
- React Router DOM
- Vite

## Objetivo del proyecto

Este proyecto fue desarrollado como práctica para fortalecer conocimientos de desarrollo front end, especialmente en:

- Componentes reutilizables
- Props
- Estado con useState
- Renderizado condicional
- Ruteo con React Router
- Estructuración de interfaces
- Experiencia de usuario en una interfaz e-commerce
